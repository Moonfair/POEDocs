# 守望石补充资料

## 作用

【备注】：仅分析前缀，后缀都一样

+ [铂金守望石](https://poedb.tw/cn/Platinum_Haewark_Hamlet_Watchstone)
  + 只提供各赛季内容的加成
+ [钛金守望石](https://poedb.tw/cn/Titanium_Haewark_Hamlet_Watchstone)
  + 补充了铂金守望石未覆盖到的赛季内容
  + 同时提供了大师任务相关的加成
+ [铬金守望石](https://poedb.tw/cn/Chromium_Haewark_Hamlet_Watchstone)
  + 提供了**大量**通货相关的加成
  + 提供了精华、瓦尔相关的加成

## 如何购买守望石

+ 每个地区有自己对应的守望石，不要买错了；比如你买了海沃克村的守望石，是插不到未来之律上的。
+ 守望石的命名规则是：
  + 【阶级】+ 【地区】+ WatchStone
+ 比如海沃克村的铂金守望石就是：
  + Platinum Haewark Hamlet Watchstone
+ 可以对照下边的对照表拼出自己想要的守望石，输入市集购买
